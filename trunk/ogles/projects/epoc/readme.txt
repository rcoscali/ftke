This file contains instructions to build ogles library and test application using 
Symbian SDK (S60 or UIQ). 

From command line go to ogles\projects\epoc folder, and run the following command:

1. S60 emulator
make.bat s60 wins udeb

2. S60 phone
make.bat s60 armi urel

3. UIQ emulator
make.bat uiq wins udeb

4. UIQ phone
make.bat uiq armi urel
