Name

glLoadPaletteFromModelViewMatrixOES - copies the current model view matrix to a matrix in the current matrix palette

C Specification

void glLoadPaletteFromModelViewMatrixOES(void)

Description

glLoadPaletteFromModelViewMatrixOES copies the current model view matrix to a matrix in the current matrix palette, as specified by glCurrentPaletteMatrix.
